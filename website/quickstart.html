Quick start

Print your first labels in about 10 minutes

Install the apps once, then pick your path: pull cable data straight from Vectorworks ConnectCAD, or design a label from scratch. Both finish at the same print window.

1 · Install VectorLabel

VectorLabel is a small suite of macOS apps: a background Engine that owns your printers and lives in the menu bar, an Auto Print helper that opens the print window when new data arrives, and two label Designers. The installer sets up all of them.

  1. Download the latest release from the GitHub Releases page (the .pkg installer is easiest).
  2. Run the installer and follow its panes — it installs all four apps.
  3. Launch VectorLabel Engine. Its icon appears in your menu bar; that icon is your home base.
  4. Plug in your printer by USB (or have its IP address ready for a network printer).
macOS Gatekeeper

Official releases are signed and notarized and open normally. If macOS ever says an app is from an “unidentified developer”, right-click the app → Open the first time to approve it. Requires macOS 14 (Sonoma) or later; VectorLabel is Mac-only.

Connect your printer

  1. Click the menu-bar icon → Preferences… → Printers.
  2. USB: click Scan USB if your printer isn’t already listed. Network: type the host/IP, choose the model, and click Add.
  3. If macOS asks to allow Documents access or notifications, allow them.
Tip

Most-mature path today is a Brady M611 (USB or network) or M610 (USB). Brother P-touch drivers are included but not yet hardware-confirmed in alpha — see the FAQ.

2 · Path A — print from Vectorworks

If your cable schedule lives in a Vectorworks ConnectCAD drawing, VectorLabel can label every wire with no re-typing. You install two small plug-in commands once, then export whenever you need labels.

Install the two plug-ins (one time)

  1. In Vectorworks: Tools → Plug-ins → Plug-in Manager.
  2. Click New Command, set Language: Python, and name it exactly Export Selected Circuits to VectorLabel. Paste in the full contents of export_selected.py.
  3. Click New Command again and create Export All Circuits to VectorLabel from export_all.py.
  4. Tools → Workspaces → Edit Current Workspace, drag both commands into a menu, and save.

Export & print

  1. Make sure VectorLabel is running on your Mac and “Auto-open print window when an export is detected” is on in its Preferences.
  2. In Vectorworks, save your file (its name becomes the export folder). Select the circuits you want — or switch to the layer you want to label.
  3. Run Export Selected Circuits (for your selection) or Export All Circuits (for the whole layer). The export is silent — nothing pops up in Vectorworks.
  4. Switch to your Mac: the print window opens automatically, one row per cable end.
  5. Pick a template and printer, review/filter the rows, and Print.
Tip

Changed the drawing? Just run the export again — a fresh export is created and the print window reopens with the new data.

3 · Path B — design a label yourself

No CAD required. Use the Custom Designer for one-off labels (with an optional database mode for batches), or the Template Designer for reusable, data-bound templates.

  1. Open a designer from the Engine menu (Open Custom Designer / Open Template Designer).
  2. Click Supply in the top bar and pick your label stock. The canvas resizes to it. For continuous tape, set a Length or click ⤢ Fit.
  3. Add objects from the left Add sidebar — Text, Barcode, Image, Symbol, lines and shapes — and edit each in the right-hand properties panel.
  4. For data-driven labels, set a text or barcode object’s source to a field or a formula (see data binding).
  5. Save your work, then print from the print window — choose printer, confirm the supply shows a green ✓, and go.
Handy keys

⌘D duplicate · ⌘C/⌘V copy & paste · Delete remove the selected object.
