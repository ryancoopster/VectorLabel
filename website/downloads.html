Get VectorLabel for macOS

Every release, and what changed in each. Download the installer, open it, and follow the wizard — it installs all four apps into /Applications/VectorLabel/. Then launch VectorLabel Engine for the menu-bar icon.

1.3.1

Pre-release · open alpha July 2, 2026

Fixes the blank “Settings” window on launch.

Download installer (.pkg) Release notes & other formats

Requires macOS 14 (Sonoma) or later, on an Apple Silicon Mac. The installer still lets you continue on an older macOS, but warns that it may not run correctly.

Changed / fixed

  • An empty "… Settings" window could open and stay on screen when any of the four apps launched or activated (seen on macOS 26). It's now closed the instant it appears.

1.3.0

Pre-release · open alpha July 2, 2026

Auto-update polish.

Download installer (.pkg) Release notes & other formats

Requires macOS 14 (Sonoma) or later, on an Apple Silicon Mac. The installer still lets you continue on an older macOS, but warns that it may not run correctly.

Changed / fixed

  • The first-launch "how should VectorLabel check for updates?" prompt now defaults to every 7 days (was: on every launch). Every option can still be chosen, and changed later in Preferences ▸ Updates.
  • Fixed an empty "VectorLabel Engine Settings" window that could appear when the update prompts surfaced.

1.2.0

Pre-release · open alpha July 1, 2026

Tables, auto-update, and cloud-file support.

Download installer (.pkg) Release notes & other formats

Requires macOS 14 (Sonoma) or later, on an Apple Silicon Mac. The installer still lets you continue on an older macOS, but warns that it may not run correctly.

Added

  • Table object in both designers — grids of data-bindable text cells with per-cell formatting, merged cells (merge/split), row/column resizing with locks, cell copy/paste, and right-click clear commands. Tables print identically everywhere.
  • Auto-update: the Engine checks GitHub for new versions — on launch, every N days, or manually (Preferences ▸ Updates) — and offers Update Now / Remind Me Tomorrow / Don’t Update with the release notes.
  • Online-only cloud files (Dropbox / iCloud / OneDrive) now download automatically before opening, with a cancellable progress popup — everywhere a file can be opened.

Changed / fixed

  • Double-clicking a table cell reliably starts editing (every cell type); bigger stepper buttons and ↑/↓ stepping in the object settings panel.
  • The installer warns (instead of blocking) on macOS older than 14.

1.1.0

Pre-release · open alpha July 1, 2026

First public release — the full four-app suite (Engine, Auto Print, Template Designer, Custom Designer).

Download installer (.pkg) Release notes & other formats

Requires macOS 14 (Sonoma) or later, on an Apple Silicon Mac. The installer still lets you continue on an older macOS, but warns that it may not run correctly.

Added

  • Design + print wire / cable / asset / panel / patch labels on a true-to-size canvas — text, barcodes, QR, DataMatrix, images, symbols and shapes.
  • Vectorworks ConnectCAD integration — export circuits and the print window opens with your records loaded.
  • Bind a CSV or Excel file (one label per row); spreadsheet-style formulas that match in preview and print.
  • Tabs across the print window and both designers — work on several labels at once.
  • Open Brady .BWT and Brother P-touch .lbx templates (auto-converted).
  • Brady M610 & M611 and Brother P-touch over USB & network, with live status and cassette auto-detection on the M611.
  • Editable supply catalog and per-printer settings (cut, orientation, calibration, feed-to-clear).

Changed

  • Auto-scale text never truncates — it shrinks the font until the whole value fits, instead of clipping to a “…”.

Fixed

  • The light / dark / auto appearance choice now relays across the whole suite — change it in the Engine and Auto Print and both designers switch too.

Open alpha: the Brady M611 is hardware-validated; the Brady M610 cut behavior and the Brother P-touch drivers are built but not yet hardware-confirmed.
