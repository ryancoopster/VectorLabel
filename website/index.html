One suite, four apps

Everything from drawing to printed label.

VectorLabel is a coordinated set of macOS apps. Design templates, watch for new exports, and drive your printers — each piece does one job well.

Engine

The print engine and menu-bar hub. Owns the printers, manages the queue, detects cassettes and publishes live status.

AutoPrint

Watches your Vectorworks export folder. The moment a new export lands, the print window opens with your records loaded.

Template Designer

Build reusable label templates, bind fields to your circuit data, and reuse them across every project.

Custom Designer

Design one-off labels with no template binding — full layout freedom, plus a database mode for browsing and editing records inline.

The workflow

From ConnectCAD circuit to printed label.

Two Python commands live inside Vectorworks. Select your circuits, run the command, and VectorLabel takes it from there — automatically.

Select in Vectorworks

Pick circuits in ConnectCAD, or export every circuit on the layer.

Export silently

A clean CSV of names, sockets, cables and racks lands in your project folder.

Auto-loaded

AutoPrint spots the file and opens the print window — records ready, preview live.

Print & cut

Choose a template and printer, hit print — with live per-label progress.

Built for the rack room

Designed for accurate, barcode-ready labels.

Every feature is aimed at one thing: getting the right text and codes onto the right label, fast, with no manual re-keying.

Visual template designer

Drag text, barcodes, images, symbols, lines, rectangles and arrows onto a true-to-size canvas. Rotate anything; auto-scale text to fit.

15 barcode & 2-D codes

QR, Data Matrix, Aztec, Micro QR, PDF417, Code 128, GS1-128, Code 39/93, EAN/UPC and more — regenerated per record at print time.

Field & formula binding

Bind any element to a data column, or compute it with Excel-style formulas — IF, LEFT, MID, UPPER, concatenation and more.

Filter, sort & presets

Print only the labels you need — filter by field, sort by column or rack/RackU, and save filter-and-sort presets for next time.

Inline record editing

Spot a typo? Click a cell in the record browser, fix it, and the change syncs straight back to the export — no context switch.

Editable supply catalog

A full catalog of Brady and Brother supplies — sizes, part numbers, quantities and buy links — all editable in-app, no code needed.

Live print progress

A per-label progress bar, responsive mid-run cancel, and a Recent Prints history you can reprint from in one click.

Import existing templates

Migrate from Brady Workstation .BWT or Brother P-touch .lbx files — geometry, text and barcodes come across intact.

Per-printer calibration

Print a calibration grid, dial in the offset once, and every label lands perfectly aligned — settings survive app updates.

Design once, print hundreds

Your data flows into the label automatically.

Map each part of the label to a column from your export — or build it with a formula. Then every record prints with its own text and its own barcode, no manual edits.

=IF(Number<>"",Number&" - "&Cable,Cable)

Native Vectorworks integration

It lives inside Vectorworks.

Install two Python commands through the Plug-in Manager and they appear right in your menus. They read ConnectCAD circuits directly — device names, sockets, connectors, cable numbers, racks and RackU — and write a clean CSV.

No copy-paste, no exporting to Excel and reformatting. The data your drawing already holds becomes the data on your labels.

Export runs silently — VectorLabel detects the new file and opens automatically.
